# bongésző
Egy alapszintű böngésző.

Automatikusan kiegészíti a weboldalak címét (pl.: google.hu -> https://google.hu)
Ha nincs a keresett szövegben pont, akkor a keresett szöveget megnyitja google keresés formájában.
